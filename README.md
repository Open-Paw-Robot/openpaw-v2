# 🐾 OpenPaw Robot — V2 (Prototype)

Generation 2 hardware of the **OpenPaw / PawMe** open-source family companion robot.

## 📁 Repository Layout

| Folder | Contents |
|:---|:---|
| `Mechanical/` | CAD, STLs & mechanical design files |
| `Electrical/` | Schematics & PCB design files |
| `Firmware/` | Firmware source & test sketches |
| `Design/` | Industrial design & product renders |
| `Docs/` | Research & product documentation |

> 📦 V2 archive files are being collected — folders are placeholders until the historical files are imported.
